27. ada yg tau jawaban soal di tvri yg sma/k hari ini tgl 16 april 2020​


Jawaban soal di TVRI yang SMA/K tanggal 16 april 2020 tentang trigonometri. Soal latihan yang diberikan terdiri dari 2 soal. Pada segitiga siku-siku memiliki sisi depan sudut (de), sisi samping sudut (sa) dan sisi miring (mi). Berikut rumus perbandingan trigonometri pada segitiga siku-siku untuk sudut α.

sin α = [tex]\frac{de}{mi}[/tex] ⇒ cosec α = [tex]\frac{mi}{de}[/tex] cos α = [tex]\frac{sa}{mi}[/tex] ⇒ sec α = [tex]\frac{mi}{sa}[/tex] tan α = [tex]\frac{de}{sa}[/tex] ⇒ cotan α = [tex]\frac{sa}{de}[/tex]

Pembahasan  

Soal nomor 1

Perhatikan segitiga ABC siku siku di titik B. AC = 13 cm, BC = 5 cm, AB = 12 cm. Tentukan perbandingan trigonometri

Jawab

a. untuk sudut A

sisi depan sudut A = de = BC = 5   sisi samping sudut A = sa = AB = 12   sisi miring = mi = AC = 13  

Jadi  

sin A = [tex]\frac{de}{mi} = \frac{BC}{AC} = \frac{5}{13}[/tex]  cos A = [tex]\frac{sa}{mi} = \frac{AB}{AC} = \frac{12}{13}[/tex]  tan A = [tex]\frac{de}{sa} = \frac{BC}{AB} = \frac{5}{12}[/tex]  cosec A = [tex]\frac{mi}{de} = \frac{AC}{BC} = \frac{13}{5}[/tex]  sec A = [tex]\frac{mi}{sa} = \frac{AC}{AB} = \frac{13}{12}[/tex]     cotan A = [tex]\frac{sa}{de} = \frac{AB}{BC} = \frac{12}{5}[/tex]  

b. untuk sudut C

sisi depan sudut C = de = AB = 12   sisi samping sudut C = sa = BC = 5   sisi miring = mi = AC = 10  

Jadi  

sin C = [tex]\frac{de}{mi} = \frac{AB}{AC} = \frac{12}{13}[/tex]  cos C = [tex]\frac{sa}{mi} = \frac{BC}{AC} = \frac{5}{13}[/tex]  tan C = [tex]\frac{de}{sa} = \frac{AB}{BC} = \frac{12}{5}[/tex] cosec C = [tex]\frac{mi}{de} = \frac{AC}{AB} = \frac{13}{12}[/tex]    sec C = [tex]\frac{mi}{sa} = \frac{AC}{BC} = \frac{13}{5}[/tex]    cotan C = [tex]\frac{sa}{de} = \frac{BC}{AB} = \frac{5}{12}[/tex]  

Soal nomor 2

Perhatikan segitiga PQR siku-siku di titik Q panjang sisi PQ = 8 cm, PR = 10 cm. Tentukan perbandingan trigonometri

Jawab

Mencari panjang QR adalah

QR = [tex]\sqrt{PR^{2} - PQ^{2}}[/tex]

QR = [tex]\sqrt{10^{2} - 8^{2}}[/tex]

QR = [tex]\sqrt{100 - 64}[/tex]

QR = [tex]\sqrt{36}[/tex]

QR = 6  

a. untuk sudut P

sisi depan sudut P = de = QR = 6 sisi samping sudut P = sa = PQ = 8 sisi miring = mi = PR = 10

Jadi

sin P = [tex]\frac{de}{mi} = \frac{QR}{PR} = \frac{6}{10} = \frac{3}{5}[/tex]    cos P = [tex]\frac{sa}{mi} = \frac{PQ}{PR} = \frac{8}{10} = \frac{4}{5}[/tex]    tan P = [tex]\frac{de}{sa} = \frac{QR}{PQ} = \frac{6}{8} = \frac{3}{4}[/tex]    cosec P = [tex]\frac{mi}{de} = \frac{PR}{QR} = \frac{10}{6} = \frac{5}{3}[/tex]    sec P = [tex]\frac{mi}{sa} = \frac{PR}{PQ} = \frac{10}{8} = \frac{5}{4}[/tex]    cotan P = [tex]\frac{sa}{de} = \frac{PQ}{QR} = \frac{8}{6} = \frac{4}{3}[/tex]

sisi depan sudut R = de = PQ = 8 sisi samping sudut R = sa = QR = 6 sisi miring = mi = PR = 10

Jadi
