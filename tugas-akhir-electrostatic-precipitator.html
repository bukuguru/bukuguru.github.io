3. Electrostatic force between two charged object is F. The distance between them is 10 mm. If the distance between them becomes 40 mm, determine the electrostatic force between them!


Answer: F1/16 = F/16

Description:

Electrostatic force = F

F = (k.q1.q2)/r²

First condition distance between 2 charge object = 10mm = 0,01 m

Second condition distance between 2 charge object = 40mm = 0,04 m

For first condition

F1 = F = k.q1.q2/(0.01)² = kq1.q2/(0.0001)

F1 = F = 10⁴.k.q1.q2

For second condition

F2 = k.q1.q2/(0.04)² = k.q1.q2/0.0016

F2 = 10⁴k.q1.q2/16

10⁴.k.q1.q2 = F1

Then

F2 = F1/16

F2 = F/16

So the electrostatic force in 40mm is F1/16 = F/16

9. Adakah kilat merupakan fenomena yang menghasilkan cas-cas elektrostatik? Is lightning a phenomena that can produce electrostatic charges? indian kilat ​


Jawaban:

Yes, lightning is a phenomenon that can produce electrostatic charges. When a lightning bolt passes through the air, it can transfer electrons from one place to another, creating a static electric charge. This charge can build up on objects near the lightning, such as trees or buildings, and can be discharged as a static electric shock when touched. So, in a sense, lightning can be thought of as a natural source of electrostatic charges.

15. The electrostatic force between two charged object is 0,8 N when the distance between them is 3,5 mm. Determine the electrostatic force if the distance between them becomes 7 mm!


Jawaban:

0,2 N

Penjelasan:

F = k q1 q2 / r²

F1/F2 = r2²/r1²

0,8/F2 = 7²/3,5² = 4

F2= 0,8/4 = 0,2 N
