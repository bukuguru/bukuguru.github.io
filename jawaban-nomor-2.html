{Nomor 2}

Menurut rumus teorema phytagoras pada segitga siku-siku, berlaku:

c² = a² + b²  

dimana:  

c = sisi miring a = alas b = tinggi

Soal:

c = 26 cm

b = 24 cm

a = ...?

Jadi:  

c² = a² + b²  

26² = a² + 24²

676 = a² + 576

676 - 576 = a²

100 = a²

√100 = a
