4. Tolong DijawabMateri : LogaritmaKelas. : 10 SMK


a.
²log45 = ²log(9 x 5)
= ²log(3² x 5)
= ²log3² + ²log5
= 2. ²log3 + ²log5
= 2.a + b

b.
²⁵log36 = ²log36 / ²log25
= ²log(9x4) / ²log5²
= ²log(3² x 2²) / ²log5²
= (²log3² + ²log2²) / ²log5²
= (2. ²log3 + 2. ²log2) / 2 . ²log5
= (2.a + 2.1) / 2.b
 = 2(a + 1) / 2b
= (a+1) / b

c.
³⁰log150 = ²log150 / ²log30
= ²log(2 x 3 x 25) / ²log(2 x 3 x 5)
= ²log(2 x 3 x 5²) / ²log(2 x 3 x 5)
= (²log2 + ²log3 + 2 . ²log5) / (²log2 + ²log3 + ²log5)
= (1 + a + 2.b) / (1 + a + b)

d.
¹⁰⁰log50 = ²log50 / ²log100
= ²log(25 x 2) / ²log(25 x 4)
= ²log(5² x 2) / ²log(5² x 2²)
= (²log5² + ²log2) / (²log5² + ²log2²)
= (2 . ²log5 + ²log2) / (2. ²log5 + 2 . ²log2)
= (2 . b + 1) / (2 . b + 2 . 1)
= (2b + 1) / (2b + 2)

e.
⁴log75 = ²log75 / ²log4
= ²log(25 x 3) / ²log2²
= (²log5² + ²log3) / 2 . ²log2
= (2 . ²log5 + ²log3) / 2 . 1
= (2 . b + a) / 2
= (a + 2b) / 2

30. kak minta tolong donk KIMIA kelas 10 SMK​


Jawaban:

A dan C

Penjelasan:

Konfigurasi elektron tiap unsur:

A = 1s2 2s2 2p4 => golongan VIA

B = 1s2 2s2 2p6 3s1 => golongan IA

C = 1s2 2s2 2p6 3s2 3p5 => golongan VIIA

D = 1s2 2s2 2p6 3s2 3p6 => golongan VIIIA

Ikatan kovalen adalah ikatan sesama non logam, golongan VIIIA termasuk golongan gas mulia yg bersifat sangat sulit bereaksi / non reaktif jadi sukar membentuk ikatan

Jadi opsi yg tepat adalah A dan C

32. kak minta tolong donk KIMIA kelas 10 SMK​


Jawaban:

d. Q dan R

Penjelasan:

Unsur yg bersifat reaktif => unsur logam golongan IA / IIA dengan unsur non logam dari golongan VIIA

Dalam konfigurasi elektron, jika unsur berakhir di subkulit s => I dan IIA tergantung elektron valensi

Jika unsur berakhir di subkulit p maka golongan unsur tersebut adalah jumlah elektron valensi antara subkulit s dan p

Kecuali Helium dgn konfigurasi elektron 1s2 => golongan VIIIA

P = Golongan VIIIA

Q = Golongan IIA

R = Golongan VIIA

S = Golongan VIIIA

Jadi yang paling reaktif adalah Q dan R
