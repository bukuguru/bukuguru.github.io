31. bahasa jawa kelas 7 smp:-Tembung lingga ing basa indonesia diarani?​


Wangsulan

➡️ Tembung lingga ing basa indonesia diarani kata dasar.

Penjelasan

a. Tembung Lingga

Tembung lingga iku tembung kang durung owah saka asale utawa isih wutuh. Ing basa Indonesia, diarani kata dasar. Tegese durung owah iku durung diwenehi ater-ater, seselan, panambang, lan pangrangkep.

Tuladha : tulis, dolan, jaluk, gambar, takon, turu, pacul, adus, ukur, padhang, lsp.

b. Tembung Andhahan

Tembung andhahan iku tembung kang wis owah saka asale utawa linggane. Owahe bisa amarga diwenehi ater-ater, seselan, panambang, utawa dirangkep. Tembung lingga kang wis diowahi iku bakal malih dadi tembung andhahan.

Tuladha :

1) madhangi = m + pandhang + i

2) pitakon = pi + takon + an

3) dolanan = dolan + an

4) dijaluk = di + jaluk

5) ngadusi = ng + adus + i

6) nggoleki = ng + golek + i

7) tinulis = tulis + in

8) gegambaran = ge + gambar + an

9) macul = m + pacul

10) ngukur = ng + ukur
