Ada dua rumus yang bisa digunakan untuk menghitung luas permukaan prisma segitiga yaitu:

Rumus 1

Luas permukaan prisma segitiga = (2 x luas alas) + luas selimut

Luas selimut artinya luas seluruh bidang tegak prisma segitiga, yaitu 3 x luas satu bidang tegak.

Rumus 2

Luas permukaan prisma segitiga = (2 x luas alas) + (keliling alas x tinggi prisma)

Karena alas prisma segitiga merupakan bangun ruang segitiga maka rumus luas alasnya adalah 1/2 x alas x tinggi.

Sedangkan, rumus untuk menghitung volume prisma segitiga yaitu:

Volume prisma segitiga = luas alas x tinggi

Sebuah benda berbentuk prisma segitiga memiliki tinggi 3 m, tinggi bidang alas 1,5 m, dan panjang bidang alas 2 m. Berapa luas permukaannya?

Penyelesaian:

Dari soal itu, terdapat informasi tentang ukuran tinggi, tinggi bidang alas, dan panjang bidang alas.

Luas permukan prisma segitiga = 2 x luas alas + luas selimut

= 2 x (1/2 x 2 x 1,5) + 3 x (2 x 3)

= 2 x 1,5 + 3 x 6

= 3 + 18

= 21 m2

Volume permukaan prisma segitiga = luas alas x tinggi

= (1/2 x 20 x 10) x 15

= 100 x 15

= 1500 cm3
