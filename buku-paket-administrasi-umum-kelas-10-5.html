9. radin artinya buku paket b.jawa kelas 5 ajar basa Jawa​


Radin iku artine rata. Radin iku basa kramane saka tembung ngoko rata.

Pembahasan

Bahasa merupakan sarana yang dapat kita gunakan dalam percakapan sehari-hari. Bahasa di setiap daerah memiliki keberagaman. Bahasa daerah dipergunakan oleh masyarakat daerah untuk melakukan percakapan dalam kehidupan sehari-hari. Misalnya pada soal tersebut merupakan bahasa Jawa, yaitu bahasa yang umumnya dipergunakan oleh masyarakat Jawa dalam kehidupan sehari-hari.

Bahasa Jawa memiliki tingkatan dalam penggunaannya. Berikut ini yang merupakan tingkatan dari bahasa Jawa:

Basa ngoko merupakan bahasa yang digunakan untuk percakapan antara yang muda misal dengan teman sebaya, orang yang sudah akrab, ataupun digunakan oleh orang tua terhadap yang muda.Basa krama merupakan tingkatan basa yang sopan dan biasa digunakan untuk melakukan percakapan terhadap orang yang lebih tua atau yang dihormati.

Pada soal tersebut menanyakan tentang arti kata radin. Radin merupakan basa krama dari basa ngoko rata. Namun kata radin terdapat juga dalam basa kawi yang memiliki arti suci atau bersih dibersihkan dari noda.

Secara sederhana, mosi dapat diartikan sebagai hasil penajaman isu dari sebuah tema yang dipilih untuk debat.

Salah satu manfaat yang dapat kita peroleh dari kegiatan debat adalah kemampuan untuk mempertahankan pandangan yang kita miliki tentang suatu isu menggunakan data-data dan fakta kredibel yang mendukung pernyataan tersebut, alih-alih menggunakan pendapat pribadi tanpa dasar.

Dalam sebuah debat, terdapat tiga pihak yang menjalankan peran masing-masing. Ketiga pihak tersebut masing-masing mencakup pihak pro atau mendukung, kontra atau oposisi atau menentang, serta seorang moderator yang berfungsi bukan hanya sebagai pemandu jalannya debat, tetapi juga untuk menggali pendapat lebih jauh serta menghindarkan debat dari terjadinya diskusi yang tidak perlu serta bentrokan akibat perbedaan pendapat.

26. siswa kelas 4 berjumlah 35 anak kelas 5 berjumlah 32 anak dan kelas 6 berjumlah 33 anak Setiap anak akan mendapatkan 8 buku paket dari sekolah Berapa jumlah buku paket yang akan dibagikan​

Penjelasan dengan langkah-langkah:

Di ketahui :

siswa kelas 4 berjumlah 35 anak

siswa kelas 5 berjumlah 32 anak

siswa kelas 6 berjumlah 33 anak

Setiap anak mendapat 8 buku paket

Di Tanya :

Berapa jumlah buku paket yang akan di bagikan

Jawab :

Jumlah buku paket =

(Jumlah siswa kelas 4 + Jumlah siswa kelas 5 + Jumlah siswa kelas 6) × buku paket yang akan di bagikan setiap anak

Jumlah buku paket =

(35 + 32 + 33) × 8 = ...

Jumlah buku paket = 100 × 8

= 800

Jadi, jumlah buku paket yang akan di bagikan adalah 800 buku paket

1.Tentukan akar persamaan berikut.

a. 3x² - 12 = 0

b. x² + 7x + 6 = 0

c. -3x² - 5x + 2 = 0

Persamaan kuadrat adalah persamaan dengan variabel berpangkat maksimal 2 dengan bentuk umum persamaan kuadrat ax² + bx + c = 0.

Pembahasan

Menentukan akar persamaan kuadrat dengan menggunakan cara memfaktorkan

a. 3x² - 12 = 0

3x² - 12 = 0

3 (x² - 4) = 0

3 (x² - 2²) = 0      [selisih kuadrat a² - b² = (a +b) (a - b)]

3 (x + 2) (x - 2) = 0

x + 2 = 0

    x₁ = -2

atau

x - 2 = 0

   x₂ = 2

HP = {-2, 2}

b. x² + 7x + 6 = 0

p × q = 6 = 1 × 6

p + q = 7 = 1 + 6

x² + 7x + 6 = 0

(x + 6) (x + 1) = 0

x + 6 = 0

    x₁ = -6

atau

x + 1 = 0

   x₂ = -1

HP = {-6, -1}

c. -3x² - 5x + 2 = 0

-3x² - 5x + 2 = 0   kesemua dikali negatif

3x² + 5x - 2 = 0

a × c = 3 × -2 = -6

p × q = -6 = 6 × -1

p + q = 5 = 6 + -1

3x² + 5x - 2 = 0

3x² + 6x - 1x - 2 = 0

3x (x + 2) - 1(x + 2) = 0

(3x - 1) (x + 2) = 0

3x - 1 = 0

    3x = 1

     x₁ = 1/3

atau

x + 2 = 0

    x₂ = -2

HP = {-2, 1/3}

(x + 1)(x - 1) = 0

x + 1 = 0 atau x - 1 = 0

x = -1 atau x = 1

HP = {-1, 1}

4x² + 4x + 1 = 0

x² + ⁴/₄ x + (⁴/₈)² = -¹/₄ + (⁴/₈)²

(x + ¹/₂)² = -¹/₄ + (¹/₂)²

(x + ¹/₂)² = -¹/₄ + ¹/₄

(x + ¹/₂)² = 0

x + ¹/₂ = 0

x = -½

HP = -½

c. - 3x² - 5x + 2 = 0   (Memfaktorkan)

- 3x² - 5x + 2 = 0  (dikali negatif)

3x² + 5x - 2 = 0

(3x - 1) (x + 2) = 0

3x - 1 = 0

    x₁ = 1/3

atau

x + 2 = 0

    x₂ = -2

HP = {-2, 1/3}

d. 2x² - x - 3 = 0  (memfaktorkan)

2x² - x - 3 = 0

(2x - 3)(x +1) = 0

x = 3/2 ataux = -1

HP = {-1, 3/2}

e. x² - x + ¼ = 0  (Rumus abc)
